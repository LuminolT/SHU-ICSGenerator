# SHU-ICSGenerator
完工啦，快乐

## 1 使用方法

1.  在教务处官网的课表查询处（[网址](http://xk.autoisp.shu.edu.cn/StudentQuery/QueryCourseTable)）复制表格，粘贴入excel样表（请注意行列对应）；
2.  运行`run.exe`；
3.  将`output.ics`发送到手机，在默认浏览器中打开，导入日历；
4.  是不是很简单，来试试看。
